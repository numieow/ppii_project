Code de bonne conduite avec une application Flask :

On met dans templates tous les fichiers html de l'application
On met dans static/css tous les fichiers css, et dans static les images que l'on va utiliser (si images utilisées il y a )
On met dans "scripts" tous les fichiers javascripts pour notre appli (doute sur le fait qu'on en ait)
On met l'application dans le répertoire, ainsi que les bases de données
